# 🎉 VS Code Workspace Projesi

Bu proje, sıfırlanmış ve 50 hakkı olan yeni bir VS Code workspace'idir.

## 📁 Proje Yapısı

```
.
├── .vscode/
│   ├── settings.json      # VS Code ayarları
│   ├── extensions.json    # Önerilen extension'lar
│   ├── launch.json        # Debug konfigürasyonu
│   └── tasks.json         # Task konfigürasyonu
├── index.js               # Ana uygulama dosyası
├── package.json           # Proje konfigürasyonu
└── README.md              # Bu dosya
```

## 🚀 Kurulum

1. **Bağımlılıkları yükleyin:**
   ```bash
   npm install
   ```

2. **Uygulamayı başlatın:**
   ```bash
   npm start
   ```

3. **Geliştirme modunda çalıştırın:**
   ```bash
   npm run dev
   ```

## 🌐 Kullanım

Uygulama başlatıldıktan sonra `http://localhost:3000` adresinden erişebilirsiniz.

## 📋 VS Code Özellikleri

### Ayarlar (.vscode/settings.json)
- Font boyutu: 14px
- Tab boyutu: 4 space
- Otomatik kaydetme
- Dark tema
- Bracket renklendirme

### Önerilen Extension'lar
- TypeScript desteği
- Prettier (kod formatlama)
- ESLint (kod kalitesi)
- Tailwind CSS
- Git entegrasyonu

### Debug Konfigürasyonu
- Node.js debugging
- Chrome debugging

### Task Konfigürasyonu
- npm install
- npm start
- npm run build
